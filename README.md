# AI Software Engineer Assignment (JS/TS)

This repository contains a small HTTP client implementation with OAuth2 support.

**Node Version**: v22.19

## How to run tests locally

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Run tests**:
    ```bash
    npm test
    ```

## How to build and run tests with Docker

1.  **Build the Docker image**:
    ```bash
    docker build -t eskalate-test .
    ```
2.  **Run the tests in Docker**:
    ```bash
    docker run eskalate-test
    ```
